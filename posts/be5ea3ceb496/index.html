手把手教你在Windows中安装DokcerDesktop

前提

开启虚拟化技术。

查看

打开任务管理器。

可以看到底下的虚拟化已启用。

下载docker

我们打开docker的官网：Docker: Accelerated Container Application Development

点击Download Docker Desktop，并选择自己电脑系统对应的安装包。

安装WSL2

github下载地址：GitHub - microsoft/WSL: Windows Subsystem for Linux

安装docker

双击安装包进行安装。不要管，直接默认点击OK。


正在安装ing

按照提示重启Windows。重启之后，会出现这个界面，点击Accept。

会弹出这个界面，选择第一个，点击Finish。

继续选择：

开启Docker服务

可以看到左下角显示Engine stopped表示docker没有启动。
右键微软的图标，选择计算机管理。

双击图片中圈出的地方。



在服务里面找到Docker Desktop Service服务。


右键点击属性。


启动类型更改为自动。


点击启动，点击确定。


可以看到已经启动成功。


启动成功可以看到如下内容。

1、弹出这个界面。


2、docker正在运行。
